# Stock_selection_system

## 高效选股系统，用于选出符合自己交易模式的股票

### 1. 功能介绍
此选股系统用来
1. 选出满足自己制定的指标的股票，同时标记出满足条件的时间点
2. 使用指定的指标，在满足从该指标的股票上回测历史数据，计算收益率。
